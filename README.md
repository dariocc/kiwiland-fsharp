# Kiwiland Fsharp

An example project for http://blog.dario.nu.

Clone with

```bash
git clone https://github.com/dariocc/kiwiland-fsharp.git
```

Run console application with:

```bash
cd kiwiland-fsharp
dotnet run project KiwilandFs/Kiwiland.fsproj
```

Run tests with:

```bash
dotnet test KiwilandFs
```
